# Project "TODO"

About project
-------------
This is a classic TODO web application with basic functionality:

- add task
- update task (in progress -> complete)
- delete task
- display only completed tasks/only new tasks/all tasks


Tech stack.
--------------
- Java 17
- PostgreSQL 14
- Spring Boot 2
- Hibernate 5
- Thymeleaf
- Bootstrap
- Liquibase
- Lombok

Environment requirements.
------------------------
1. Java 17
2. Maven 3.8
3. PostgreSQL 14
